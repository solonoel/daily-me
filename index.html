🔍 Keywords — How They Work
🔍 Case-insensitive, whole-word match — searched on YouTube and filtered news sources. Drag to reorder — higher items are prioritized.
💬 Phrases: Multi-word phrases must use quotes — e.g. "White Sox", "Stock Market". Without quotes, each word is matched independently.
📋 Clauses ( , or ; ): Comma or semicolon separates independent clauses — a headline passes if it matches any clause. e.g. Bears+Stadium; Cubs+Stadium matches either team's stadium news.
➕ AND ( + ): All terms must be present — e.g. Bears+Stadium requires both words.
➗ OR ( / ): Any term matches — e.g. Russia/Russian matches either word. OR is evaluated before AND, so Russia/Russian+Ukraine means (Russia OR Russian) AND Ukraine.
🚫 Exclusions ( - ): Prefix a term with - to exclude headlines containing it — e.g. Bears+Stadium-Indiana. Exclusions apply per clause. Use quotes for phrases: -"Indiana Pacers".

Keywords tell Daily Me what to look for in headlines. Multi-word phrases must use quotes, e.g. "White Sox".
